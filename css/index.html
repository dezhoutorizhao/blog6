CSS

2022-05-15 留言 CSS

网页分为

  1. 结构:HTML
  2. 表现:CSS
  3. 行为:JS

CSS 层叠样式表

  1. 层叠：网页是一个立体的结构，只是看起来像是一个平面，其实有一定的层次结构
  2. 我们看到的只是最上面的一层
  3. 通过CSS可以为网页的每一个层设置样式
  4. 总之：CSS可以用来设置网页中元素的样式

使用CSS来修改元素的样式

第一种方式：内联样式（不推荐使用）

  1. 在标签内部通过style属性来设置元素的样式
  2. 只能在开始标签中设置,style的值即为css
  3. 名字和值之间使用冒号连接
  4. 每一个名值对以分号结尾
    <p style="color:red; font-size:20px"></p>

    不推荐使用内联样式

  5. 使用内联样式，只能对一个标签生效
  6. 如果希望影响到多个元素，必须在每一个元素中都复制一遍
  7. 后期维护的时候，如果前期复制一百遍，后期就要改一百遍，维护起来非常不方便

CSS选择器

p{color:green;}
  1. 标签名+{属性}

开发时一定一定不要使用内联样式

第二种方式：内部样式表

  1. 将样式编写到head中的style标签中
  2. 然后通过CSS的选择器来选中元素，并为其设置各种样式
  3. 可以同时为多个标签设置样式，并且修改时只需要修改一处即可全部应用
  4. 内部样式表更方便对样式进行重复使用
  5. 内部样式表只能对一个页面起作用，里面的样式不能跨页面进行复用

第三种方式：外部样式表（最佳使用方式）

可以将CSS样式编写到一个外部的CSS文件中

然后通过link标签引入外部的CSS文件

p{
    color: tomato;
    font-size: 100px;
}

< link>

  1. 通过link标签来引入外部的CSS文件
  2. 外部样式表需要通过link标签进行引入，意味着只要想使用这些样式的网页可以对其进行引用

作用：

  1. 使样式可以在不同页面进行复用，提高复用率
  2. 将样式编写到外部的CSS文件中，可以使用浏览器的缓存机制，从而加快网页的加载速度，提高用户的体验

CSS语法

和HTML不同

  1. CSS语法和HTML的语法互相隔离

注释：

  1. 和c++中的多行注释相同
    /*
abc
*/

基本语法

p{
    color: red;
    font-size: 20px;
}

选择器

  1. 通过选择器可以选中页面中的指定元素
  2. 选择器选中的是html中的标签

声明块

  1. 通过声明块来指定要为元素设置的样式
  2. 声明块由一个个的声明组成
  3. 声明是由一个个的名值对组成

名值对

  1. 一个样式名对应一个样式值，名和值之间以’:’连接，以’;’结尾
  2. 样式名值对之间的分号不能省略，最后一个名值对后面的分号可以省略（我联想到了sql语句select选中列的语法相同）

选择器

元素选择器

作用

  1. 根据标签来选中指定的元素

    语法

  2. 标签名{}
    p{} h1{} div{}

id选择器

作用

  1. 根据元素的id属性值选中一个元素

语法

#id属性值{}

注意

  1. id值只有一个，不能重复使用

class选择器（类）

  1. class是一个标签的属性，和id类似，不同的是，class可以重复使用
  2. 可以通过class属性来为元素分组
  3. 可以同时为一个元素指定多个class属性
  4. 标签中的多个class属性通过空格隔开&/nbsp;

语法

.class(属性值)

通配选择器

作用

  1. 选中页面中的所有元素

    语法

    *{属性值}

复合选择器

交叉选择器

  1. 选中同时满足多个条件的元素
  2. 多个选择器的复合

    语法

  3. 选择器1.选择器2.选择器n{}
    div.red{
    font-size: 30px;
    color: red;
}

注意

  1. 交叉选择器中如果有元素选择器，必须使用元素选择器开头
  2. 一般不会出现id选择器和其它选择器的复用，因为id选择器已经可以唯一确定一个标签，一般只是元素选择器和类选择器之间的交叉

选择器分组/并集选择器

作用

  1. 同时选择多个选择器对应的元素

语法

选择器1,选择器2,选择器n{}
  1. 逗号在此处相当于是并集符号

交叉选择器和并集选择器可以复合使用

网页中不同元素之间的关系

父子关系

  1. 直接包含子元素的是父元素（注意是直接）
  2. 直接被父元素包含的是子元素

    祖先后代元素

  3. 直接或间接包含后代元素叫祖先元素
  4. 一个元素的父元素也是祖先元素
  5. 直接或间接被祖先元素包含的元素叫做后代元素

兄弟元素

  1. 拥有相同父元素的叫兄弟元素

关系选择器

子元素选择器

作用：

  1. 选中指定父元素的指定子元素

    语法

    父元素 > 子元素
div > span {
    color: red;
}

后代元素选择器

作用：

  1. 选中指定元素的指定后代元素
  2. 范围比子选择器更大，相当于递归处理（-r）

语法

祖先 后代{

}

兄弟元素选择器

作用

  1. 选择下一个兄弟
  2. 注意必须是紧邻的下一个兄弟元素
  3. 隔开的兄弟元素不会被选中
  4. 前面的兄弟元素不会被选中，只会选中前面的兄弟元素

语法

前一个 + 下一个
 p + span{

 }
选择下边所有的兄弟
兄 ~ 弟{

}

属性选择器

title可以指定标签的标题

  1. 写的是什么，鼠标光标移到某个位置，可以显示提示文字

语法

元素选择器[属性]{
    属性
}



作用

  1. 选择含有指定属性的元素

分类

[属性名] 选择含有指定属性的元素
[属性名=属性值] 选择含有指定元素属性和属性值的元素
[属性名^=属性值] 选择属性值以指定值开头的元素
[属性名$=属性值] 选择属性值以指定值结尾的元素
[属性名*=属性值] 选择属性值中含有某值的元素
